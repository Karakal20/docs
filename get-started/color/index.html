Color System

Our color system is based on a Atlassian design color foundation that designed for our components and element styles.

Contents
  • Colors
  • Color palette

Bootstrap is supported by an extensive color system that themes our styles and components. This enables more comprehensive customization and extension for any project.

Colors

Bootstrap colors ending in -rgb provide the red, green, blue values for use in rgb() and rgba() color modes. For example, rgba(var(--bs-secondary-bg-rgb), .5).
